# Galaxy Shoes

**A futuristic sneaker e-commerce store built with React, Vite, GSAP & Framer Motion.**

🌐 **Live Site → [https://jiyagarg-10.github.io/galaxy-shoes/](https://jiyagarg-10.github.io/galaxy-shoes/)**

---

## Features

- Animated SVG sneakers with CSS filter colorways
- Smooth scroll powered by Lenis + GSAP ScrollTrigger
- Cosmic orbit animation with 5 satellite shoes
- Full e-commerce flow — cart, size picker, checkout & order confirmation
- Auto-scrolling customer reviews marquee
- Responsive — works on mobile and desktop
- Framer Motion entrance animations throughout

## Tech Stack

| Tool | Purpose |
|------|---------|
| React 19 + Vite | Frontend framework & build tool |
| GSAP + ScrollTrigger | Scroll-driven animations |
| Framer Motion | UI animations & transitions |
| Lenis | Smooth scrolling |
| React Context | Cart state management |

## Running Locally

```bash
npm install
npm run dev
```

## Deploying

```bash
npm run deploy
```

Builds and pushes to the `gh-pages` branch automatically.
